# Titanic Survival Prediction (Machine Learning)

##  Overview
This project predicts whether a passenger survived the Titanic disaster using Logistic Regression.

The focus is on:
- Data preprocessing
- Feature engineering
- Model training
- Model evaluation

---

##  Tools & Technologies
- Python
- Pandas
- Scikit-learn
- Matplotlib

---

##  Dataset
Source: Kaggle Titanic Dataset

Features used:
- Age
- Sex
- Passenger Class
- Fare
- Embarked

Target:
- Survived (0 or 1)

---

##  Implementation Steps
1. Loaded dataset using Pandas.
2. Handled missing values.
3. Converted categorical variables to numeric.
4. Selected relevant features.
5. Split dataset into training and testing sets.
6. Trained Logistic Regression model.
7. Evaluated accuracy and confusion matrix.

---

##  Model Performance
- Accuracy Score: ~75%–85% (depending on preprocessing)
- Evaluated using confusion matrix
