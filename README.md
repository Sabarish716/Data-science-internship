#  Data Science Internship Project Portfolio

This repository contains 4 completed data science projects demonstrating skills in:

- Data Analysis
- Data Cleaning
- Data Visualization
- Machine Learning
- Regression Modeling
- Recommendation Systems

Each project focuses on solving a real-world problem using Python and industry-standard tools.

---

#  Projects Included

## 1️ COVID-19 Data Analysis

###  Objective
Analyze COVID-19 trends including:
- Daily cases
- Death rates
- Recovery patterns

###  Skills Demonstrated
- Time-series data handling
- Data cleaning
- Data visualization using Matplotlib & Seaborn

### Key Outcome
Identified wave patterns, peak infection periods, and recovery improvements using line charts.

---

## 2️ Titanic Survival Prediction (Classification)

###  Objective
Build a machine learning model to predict whether a passenger survived the Titanic disaster.

###  Skills Demonstrated
- Logistic Regression
- Feature encoding
- Data preprocessing
- Model evaluation (Accuracy, Confusion Matrix)

###  Key Outcome
Achieved strong prediction accuracy by handling categorical features and optimizing model inputs.

---

## 3️ Movie Recommendation System (Content-Based Filtering)

###  Objective
Develop a recommendation system that suggests movies based on genre similarity.

###  Skills Demonstrated
- Text vectorization
- Cosine similarity
- Content-based filtering
- Similarity matrix generation

###  Key Outcome
Successfully recommended similar movies using genre-based vector comparison.

---

## 4️ Salary Prediction Model (Regression)

###  Objective
Predict salary based on:
- Experience
- Job role
- Education level
- Other structured features

###  Skills Demonstrated
- Linear Regression
- One-Hot Encoding
- Model evaluation using MSE & R²
- Feature selection

### Key Outcome
Built a regression model capable of estimating salary with measurable accuracy.
